Sass does not have semicolons or brackets vs SCSS

Variables when using something over and over again (color, padding, font) at top; [#name: value;]

nesting allows for simplification (instead of nav ul, write nav{ ul{)

@import to copy files @extend to copy a .css

partials are partial .scss used to import into main.scss and generate main.css (labelled _name.scss)

operations are possible (1px / 2px * 100%)

mixins are similar to functions and pass value in to populate multiple; @mixin default-padding($value) { padding=left: $value;} .panel {.panel {@include default-padding(10px);}

&- for parent elements (.button {&:visited {} &:hover{} &:active {}})
